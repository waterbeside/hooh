<div align="center">
  <h1>HOOH</h1>
</div>

HOOH is a back-end framework for the convenience of building api or MVC services. Use KOA2 + Typescript + redis + typeorm + nunjucks. Free from complicated configuration.

## Installation

```sh
npm init hooh <your-project-name>
```

```sh
npm install
```

## Start Application

```sh
npm run dev
```
